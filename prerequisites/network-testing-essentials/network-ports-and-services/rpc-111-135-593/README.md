# RPC (111,135,593)

**Remote Procedure Call (RPC)** allows a program on one machine to execute functions on another machine as if they were local.

{% code overflow="wrap" %}
```
RPC
R => Remote
P => Procedure
C => Call
Name itself suggest calling functions (procedure) on remote machine. 
```
{% endcode %}

Windows uses RPC extensively for:

* User enumeration
* Group enumeration
* Service management
* Scheduled tasks
* Registry access
* Domain operations
* Printer services

{% hint style="info" %}
_RPC is not a service. RPC is a communication method that many windows services uses._&#x20;
{% endhint %}
